# 中文彩票预测技能 - 修复版

## 概述
这是一个修复后的彩票预测技能，专门处理中国彩票（大乐透、双色球）的预测分析。**重要修复**：现在正确考虑了春节等节假日休市安排。

## 修复内容

### 主要问题修复
**问题**：之前的系统忽略了春节休市，导致给出错误的开奖时间。

**修复**：
1. **添加节假日判断**：包含2026年春节休市（2月14日-2月23日）
2. **正确计算开奖时间**：休市结束后找到第一个开奖日
3. **节假日提醒**：在报告中明确标注节假日状态

### 新增功能
- Node.js实现（符合此方技术偏好）
- 多种预测策略：热号、冷号、混合
- 预算计算和购彩建议
- JSON数据导出

## 快速开始

### Node.js版本（推荐）
```bash
# 预测大乐透，预算10元
node lotteryPredict.js dlt 10

# 预测双色球，预算20元
node lotteryPredict.js ssq 20
```

### Python版本
```bash
# 预测大乐透，预算10元
python lottery_predict.py dlt 10

# 预测双色球，预算20元
python lottery_predict.py ssq 20
```

## 参数说明
- **彩票类型**：
  - `dlt`：大乐透（前区35选5，后区12选2）
  - `ssq`：双色球（红球33选6，蓝球16选1）
- **预算**：整数，单位元（默认：10）

## 节假日配置
节假日配置在代码的 `HOLIDAYS` 常量中，目前包含：
- **2026年春节**：2月14日-2月23日（10天休市）

如需添加其他节假日，可修改代码中的 `HOLIDAYS` 对象。

## 技术细节

### 预测算法
1. **历史数据分析**：模拟最近30期开奖数据
2. **热号/冷号识别**：统计号码出现频率
3. **策略生成**：
   - 热号策略：选择出现频率最高的号码
   - 冷号策略：选择出现频率最低的号码
   - 混合策略：70%热号 + 30%冷号

### 开奖时间计算
1. 检查当前日期是否在节假日期间
2. 如果是节假日，跳过节假日期间
3. 找到节假日后的第一个开奖日
4. 考虑彩票的正常开奖日（大乐透：周一、三、六；双色球：周二、四、日）

## 注意事项
1. **数据来源**：当前版本使用模拟数据，实际应用时应接入真实历史数据
2. **节假日更新**：需要每年更新节假日配置
3. **风险提示**：彩票预测仅供参考，请理性投注
4. **开奖时间**：实际开奖时间以官方公告为准
