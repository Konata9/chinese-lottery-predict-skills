# Chinese Lottery Predict Skill (中国彩票预测技能)

这是一个基于 AI 的彩票预测技能，旨在通过分析历史开奖数据（双色球、大乐透等），为用户提供下一期的号码预测参考。

## 🎯 核心功能

*   **历史数据分析**：自动抓取并分析近 30-50 期的开奖记录。
*   **冷热号统计**：识别近期频繁出现（热号）和长期未出（冷号/遗漏）的号码。
*   **智能推荐**：基于统计学规律生成 1-5 组推荐号码。
*   **多源检索**：优先检索官方权威数据源（中彩网、500彩票网等），并具备自动降级策略以应对反爬虫机制。

## �🚀 使用方法

当您询问以下类型的中文问题时，该 Skill 将自动触发：

*   "帮我预测下期双色球"
*   "大乐透下期买什么号码好"
*   "分析一下最近的双色球走势"
*   "给我推荐几注大乐透，预算 20 块"

## 🛠️ 工作原理

1.  **意图识别**：识别用户想要预测的彩种（如双色球、大乐透）及预算金额。
2.  **数据获取**：
    *   **优先**：访问官方源（cwl.gov.cn, zhcw.com 等）。
    *   **兜底**：如果官方源不可用，自动转向新浪、网易等门户网站的静态历史数据页。
3.  **统计分析**：计算每个号码的出现频率和遗漏值。
4.  **生成报告**：输出包含热号、冷号、推荐组合及购彩建议的结构化报告。
